Diferenciacion numerica

descripcion

La diferenciación numérica es una técnica usada para aproximar derivadas de una función cuando no se conoce su forma analítica o solo se tienen datos discretos (tablas o mediciones).

La derivada de una función f(x) mide qué tan rápido cambia la función en un punto:

f′(x) = límh→0 f(x + h) − f(x) h

Como en los datos reales no podemos hacer que h → 0, se emplean diferencias finitas:

1.1 Diferencia hacia adelante

f′(x) ≈ f(x + h) − f(x) h

Usa el punto actual y el siguiente. Ideal para el inicio de una tabla de datos.

1.2 Diferencia hacia atrás

f′(x) ≈ f(x) − f(x − h) h

Usa el punto actual y el anterior. Ideal para el final de una tabla.

1.3 Diferencia centrada

f′(x) ≈ f(x + h) − f(x − h) 2h

Usa valores a ambos lados del punto, por lo que es más precisa.

1.4 Segunda derivada

La segunda derivada, que mide la aceleración o curvatura, se calcula como:

f′′(x) ≈ f(x + h) − 2f(x) + f(x − h)
