Trabajo 1: RECONOCIMIENTO DE FUNCIONES

Python
identificador de funciones

Se quiere crear un programa que, a partir de una funci√≥n matem√°tica ingresada por el usuario con hasta dos variables, determine cu√°les son esas variables y cuente cu√°ntas operaciones aparecen. Entre las operaciones a considerar est√°n: suma, resta, multiplicaci√≥n, divisi√≥n, potencias y multiplicaciones impl√≠citas (como 5x o xy). El programa debe analizar la funci√≥n y producir como resultado las variables detectadas, cu√°ntas variables diferentes hay y el total de operaciones identificadas.

import re

def analizar_funcion(funcion):
    funcion = funcion.replace(" ", "")
    variables = sorted(set(re.findall(r'[a-zA-Z]', funcion)))
    terminos = re.findall(r'[+-]?\d*[a-zA-Z]?\^?\d*', funcion)
    terminos = [t for t in terminos if t]
    coeficientes = []
    signos = []
    for t in terminos:
        if t.startswith('-'):
            signos.append('-')
        else:
            signos.append('+')
        c = re.findall(r'[+-]?\d+', t)
        if c:
            coef = int(c[0])
        else:
            coef = -1 if t.startswith('-') else 1
        coeficientes.append(coef)
    print(f"\nüîπ Funci√≥n ingresada: {funcion}")
    print(f"üî∏ Variables encontradas: {variables}")
    print(f"üî∏ T√©rminos: {terminos}")
    print(f"üî∏ Coeficientes: {coeficientes}")
    print(f"üî∏ Signos: {signos}")

print("=== ANALIZADOR DE FUNCIONES ALGEBRAICAS ===")

se desarrollo un programa en Python capaz de reconocer dos funciones lineales y generar sus respectivos gr√°ficos, para lo cual el limite de unidades por cada coordenada en el plano cartesiano es de 20 unidades y no se utilizaron librerias especializadas de Python.

def preparar_expresion(expr: str) -> str:
    expr = expr.replace(" ", "")          # quitar espacios
    expr = expr.replace("^", "")        # potencia
    expr = expr.replace("-x", "-1*x")     # caso -x
    expr = expr.replace("+x", "+1*x")     # caso +x
    if expr.startswith("x"):              # si empieza con x
        expr = "1*" + expr
    expr = expr.replace("x", "*x")        # poner multiplicaci√≥n
    expr = expr.replace("**x", "*x")      # corregir si se duplic√≥
    return expr

# Pedimos las dos funciones
func1 = input("Ingrese la primera funci√≥n (ejemplo: 2x+1): ")
func2 = input("Ingrese la segunda funci√≥n (ejemplo: -x+3): ")

func1 = preparar_expresion(func1)
func2 = preparar_expresion(func2)

# Definimos el rango del gr√°fico (valores de X y Y)
xmin, xmax = -20, 20
ymin, ymax = -20, 20

# Recorremos los valores de Y de arriba hacia abajo
for y in range(ymax, ymin - 1, -1):
    linea = ""

    for x in range(xmin, xmax + 1):
        try:
            y1 = eval(func1, {"x": x})
        except:
            y1 = None
        try:
            y2 = eval(func2, {"x": x})
        except:
            y2 = None

        cond1 = (y1 is not None and abs(y - y1) < 0.5)
        cond2 = (y2 is not None and abs(y - y2) < 0.5)

        if cond1 and cond2:
            linea += "#"
        elif cond1:
            linea += "*"
        elif cond2:
            linea += "o"
        elif x == 0 and y == 0:
            linea += "+"
        elif x == 0:
            linea += "|"
        elif y == 0:
            linea += "-"
        else:
            linea += " "
    print(linea)

# Leyenda
print("\nLeyenda del gr√°fico:")
print("  * = Funci√≥n 1")
print("  o = Funci√≥n 2")
print("  # = Intersecci√≥n")
print("  | = Eje Y")
print("  - = Eje X")
print("  + = Origen (0,0)")

Trabajo 4: M√âTODO DE LA SECANTE

Python
M√âTODO DE SECANTE

El m√©todo de la secante es una t√©cnica num√©rica usada para encontrar, de manera aproximada, el valor donde una funci√≥n se hace cero. En lugar de utilizar derivadas o c√°lculos complicados, este m√©todo se basa en ir probando valores y ajust√°ndolos poco a poco. La idea principal es tomar dos puntos iniciales donde la funci√≥n cambia de comportamiento y trazar una l√≠nea recta imaginaria entre ellos. Esa l√≠nea sirve como una gu√≠a para adivinar un nuevo valor que est√© m√°s cerca de la respuesta correcta. Luego, el proceso se repite usando los puntos m√°s recientes, permitiendo que el c√°lculo se vaya afinando en cada paso. El m√©todo de la secante es atractivo porque suele ser r√°pido, simple y eficiente, especialmente cuando no se puede calcular una derivada. Aunque no siempre garantiza llegar a la soluci√≥n, cuando funciona bien permite obtener aproximaciones muy precisas con relativamente pocos pasos.

Trabajo 5: M√âTODO DE NEWTON RAPHSON

Python
Sistemas Lineales

El m√©todo de Newton-Raphson es un procedimiento iterativo utilizado para encontrar aproximaciones sucesivas de las ra√≠ces reales de una funci√≥n continua y derivable. Su fundamento radica en el desarrollo del polinomio de Taylor de primer orden alrededor de un punto inicial x‚ÇÄ, lo cual permite estimar el punto donde la funci√≥n corta el eje x mediante la siguiente f√≥rmula recursiva: x‚Çô‚Çä‚ÇÅ = x‚Çô - f(x‚Çô) / f'(x‚Çô). En cada iteraci√≥n, el m√©todo utiliza la pendiente de la tangente en el punto x‚Çô para aproximarse progresivamente hacia la ra√≠z. Si la funci√≥n es suficientemente suave y el punto inicial se elige adecuadamente, la convergencia suele ser r√°pida y cuadr√°tica.

import math

# ===================================================
# M√âTODO DE NEWTON-RAPHSON (O RAPHSON SIMPLE)
# ===================================================
def newton_raphson(f, df, x0, tol=1e-6, max_iter=100):
    x = x0

    print("\n" + "="*65)
    print(" M√âTODO DE NEWTON-RAPHSON - TABLA DE ITERACIONES ")
    print("="*65)
    print(f"{'Iter':<6}{'x':<15}{'f(x)':<15}{'f\'(x)':<15}{'Error':<15}")
    print("-"*65)

    for i in range(1, max_iter + 1):
        fx = f(x)
        dfx = df(x)

        if abs(dfx) < 1e-10:
            print("‚ö†  Error: Derivada muy peque√±a. Deteniendo el m√©todo.")
            break

        x_nuevo = x - fx / dfx
        error = abs(x_nuevo - x)

        print(f"{i:<6}{x:<15.8f}{fx:<15.8f}{dfx:<15.8f}{error:<15.8e}")

        if error < tol:
            print("-"*65)
            print(f"‚úÖ Convergencia alcanzada en {i} iteraciones.")
            print("="*65)
            return x_nuevo, i

        x = x_nuevo

    print("-"*65)
    print(f"‚ö†  No se alcanz√≥ la tolerancia despu√©s de {max_iter} iteraciones.")
    print("="*65)
    return x, max_iter
